# Jettra NB Designer

Apache NetBeans Plugin for JettraWUI Visual Design.

## Features

- **Integrated Designer**: Open the Jettra Designer directly within the NetBeans IDE.
- **Drag & Drop**: Visual interface for building JettraWUI pages.
- **Project Integration**: Access your project models and classes directly from the designer.

## Installation

1. Open this project in Apache NetBeans.
2. Right-click the project and select "Install".
3. The plugin will be registered in the IDE.

## How to use

1. Go to `Window` -> `Jettra Designer`.
2. A new editor tab will open with the designer interface.
3. Use the designer to build your UI and generate the Java source code.
